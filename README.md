# 자동차 경주 게임 - TDD
## 자동차 경주 게임 요구사항
- 사용자는 이름을 쉼표 기준으로 구분해 입력
- 이름은 5자 이내
- 사용자는 몇 번의 이동을 할 것인지 입력
- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈춤
- 전진은 각 자동차 0-9의 random 값이 4이상이면 전진, 3이하면 멈춤
- 전진하는 자동차 출력할 때 이름도 같이 출력
- 우승자 출력
- 공동 우승이면 우승자 모두 출력
<br>
